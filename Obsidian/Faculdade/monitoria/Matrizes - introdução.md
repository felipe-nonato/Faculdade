## O que é?

- Tabela disposta em linhas e colunas

→ Exemplo:
A =
	[3 0]
	[2 -1]
	[7 5]

- Sempre representar por letras maiusculas.

B = 
	(3 1)
	(0 -4)

- Ordem -> numero de linhas e colunas.

	A = 3x2
	B = 2x2

- Classificação

> Matriz unitaria: Só possui um elemento.
> 
> ex: 
> A = [3]

>Matriz linha: Possui apenas uma linha.
>
>ex:
>A = [0 5 7]
>ordem: 1x3

>Matriz coluna: Possui só 1 coluna.
>
>ex:
>A = [2]
>       [3]
> ordem: 2x1

>Matriz Retangular: Apresenta os dados em forma retangular.
>
>Ex:
>A = [2 3 0]
>	   [5 9 4]
>ordem: 2x3

>Matriz Quadrada: Apresenta os dados em forma quadrada.
>
>Ex:
>A = [2 3 0]
>	   [5 9 4]
>	   [10 6 3]
>ordem: 3x3

- Elementos de uma matriz: 

	os elementos de uma matriz sempre são representados por meio de letras minusculas.

i → Linha
j → Coluna

- Diagonal principal

	Da esquerda para direita

- Diagonal Secundaria

	Da direita pra esquerda
